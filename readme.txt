A mobile app for Easy Digital Downloads

== Description ==

Access Easy Digital Downloads sales data, on the go, from an iPhone or Android device.


== Installation ==

1. Upload folder edd-mobile to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your-site/edd-mobile from your mobile device.

== Frequently Asked Questions ==

= This plugin requires Easy Digital Downloads =

You will need to install Easy Digital Downloads and create an API Key and Token for each user you want to allow access to the API. Create API keys by visiting a users profile in the WordPress Admin.
